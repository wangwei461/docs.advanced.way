## 简介

MyBatis 是一款优秀的持久层框架，一个半 ORM（对象关系映射）框架，它支持定制化 SQL、存储过程以及高级映射

`iBatis` 来源于 `internet abaits` 的组合，是一个java的持久化框架, 2010 更名为 `mybatis`
